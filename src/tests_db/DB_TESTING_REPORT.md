# Отчёт по автоматизированному тестированию базы данных

## Цель тестирования

После обновления структуры базы данных и сервисного слоя было проведено повторное автоматизированное тестирование бизнес-логики приложения CareForMe.

Цель тестирования:

* проверить корректность работы сервисного слоя;
* убедиться в правильности обработки пользовательских действий;
* проверить работу системы растений, достижений, уровней и советов;
* выявить возможные ошибки после изменений в базе данных.

---

## Используемые инструменты

* Python 3.10
* pytest
* unittest.mock
* pytest-cov

---

## Протестированные сервисы

### UserService

Проверены:

* получение статистики пользователя;
* получение текущего уровня;
* работа ежедневной серии входов;
* получение информации об уровнях;
* работа слотов растений;
* разблокировка дизайнов;
* смена горшков и леек;
* обновление пользовательской статистики.

Количество тестов: **28**

---

### FlowerService

Проверены:

* посадка растений;
* полив растений;
* обработка ошибок пользователя;
* изменение освещения;
* изменение местоположения растения;
* проверка здоровья растения;
* рост растения;
* достижение зрелости;
* восстановление погибшего растения.

Количество тестов: **34**

---

### ChallengeService

Проверены:

* получение достижений;
* расчёт прогресса достижений;
* регистрация ошибок пользователя;
* регистрация идеального выращивания;
* регистрация смерти растения;
* получение статистики достижений.

Количество тестов: **15**

---

### LevelQuestService

Проверены:

* получение заданий уровня;
* получение прогресса пользователя;
* выполнение заданий;
* повышение уровня;
* выдача наград;
* получение статуса всех уровней;
* проверка специальных условий заданий.

Количество тестов: **22**

---

### TipsService

Проверены:

* получение всех советов;
* получение позитивных советов;
* поиск советов по типу;
* получение советов для растений;
* обработка ошибок парсинга данных.

Количество тестов: **11**

## Вывод

В результате тестирования подтверждена корректная работа основных компонентов сервисного слоя приложения CareForMe.

Разработанный набор из 110 автоматизированных тестов обеспечивает проверку ключевой бизнес-логики проекта и позволяет быстро выявлять ошибки при последующих изменениях базы данных и программного кода.
